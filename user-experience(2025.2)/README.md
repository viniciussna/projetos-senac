# User Experience – Semiótica Aplicada a Interfaces Amigáveis (2025.2)

## Descrição da Atividade
Esta atividade teve como objetivo aplicar conceitos de **semiótica** na análise de interfaces digitais, compreendendo como elementos visuais comunicam significado para os usuários.

Foram trabalhados conceitos como:

- **Signo**
- **Semiose**
- **Ícone**
- **Índice**
- **Símbolo**

## Interface Analisada
Para a análise desta atividade, foi utilizada a **interface do Projeto Integrador (PI)** desenvolvida durante o curso.

A partir dessa interface, foi criado um **mapa de significação** para identificar como os elementos visuais transmitem informações e orientam a interação do usuário.

## Mapa de Significação
O mapa foi organizado em três colunas principais:

| Elemento da Interface | Tipo de Signo | Significado Percebido |
|----------------------|---------------|-----------------------|
| Elementos visuais da interface | Ícone, índice ou símbolo | Interpretação do usuário |

A análise buscou verificar se existe **coerência entre os signos utilizados na interface e a identidade visual do projeto**, avaliando a clareza e a legibilidade da comunicação visual.

## Entregável
A atividade foi apresentada em formato de **pitch**, utilizando slides desenvolvidos no **Canva**.

📄 O material completo da atividade está disponível no arquivo PDF.

## Objetivo da Atividade
- Desenvolver a capacidade de interpretar signos em interfaces digitais.
- Aplicar conceitos de semiótica no design de interfaces.
- Avaliar a coerência entre elementos visuais e experiência do usuário.
