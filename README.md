



# Projeto SENAC

Este repositório reúne as atividades, projetos e estudos desenvolvidos durante minha formação na área de **Tecnologia da Informação no SENAC**.

Aqui estão organizados materiais produzidos ao longo do curso, incluindo exercícios práticos, projetos acadêmicos, modelagens de banco de dados, protótipos de interfaces e pesquisas relacionadas ao **Projeto Integrador (PI)**.

O objetivo deste repositório é documentar minha evolução durante o curso e servir como **portfólio acadêmico**, demonstrando as habilidades adquiridas em diferentes disciplinas.

---

# Estrutura do Repositório

O conteúdo está organizado por **semestre**, separando as atividades de acordo com o período em que foram desenvolvidas.

## 1º Semestre – 2025.1

Atividades iniciais do curso, focadas em fundamentos de programação, desenvolvimento web e construção de portfólio digital.

Principais conteúdos:

* Aulas e exercícios de **Coding**
* Desenvolvimento de **portfólio digital**
* Atividades de **extensão em Frontend**
* Primeiras experiências com desenvolvimento web

---

## 2º Semestre – 2025.2

Neste semestre foram desenvolvidas atividades mais aprofundadas, incluindo **Projeto Integrador**, **Banco de Dados**, **Experiência do Usuário**, **Pesquisa científica** e **Legislação em tecnologia**.

### Projeto Integrador

Projeto desenvolvido ao longo do semestre com foco em uma **solução digital para gestão do programa de aquisição e distribuição de sementes**.

O projeto inclui:

* Protótipo de interface desenvolvido no **Figma**
* Modelagem e estrutura de **banco de dados**
* Pesquisa científica para fundamentação da solução
* Estudos sobre experiência do usuário

### Banco de Dados

Atividades relacionadas à disciplina de Banco de Dados II, incluindo:

* Modelagem de dados (MER e MR)
* Desenvolvimento de scripts SQL
* Exercícios de consultas com **JOIN**
* Projetos completos de banco de dados
* Mini-mundos modelados para diferentes cenários

Entre os projetos desenvolvidos estão:

* Modelagem de sistemas empresariais
* Sistema de gestão para editora
* Exercícios utilizando mini-mundos como Petshop e Empresa

### User Experience (UX)

Atividades voltadas para análise e desenvolvimento de interfaces digitais, incluindo:

* Estudo de **semiótica aplicada a interfaces**
* Análise de elementos visuais e seus significados
* Avaliação da legibilidade semiótica em interfaces digitais

### Pesquisa, Tecnologia e Sociedade

Atividades de pesquisa utilizando ferramentas acadêmicas para encontrar e analisar artigos científicos relacionados ao tema do Projeto Integrador.

Foi utilizada a ferramenta **Litmaps** para identificar conexões entre artigos e construir a base teórica do projeto.

### Pesquisa Científica

Desenvolvimento de pesquisa aplicada ao Projeto Integrador, incluindo:

* entrevistas com pessoas da área rural
* questionários sobre expectativas relacionadas à solução proposta
* tabulação e análise de dados
* relato de experiência e conclusões da pesquisa

### Legislação e Tecnologia da Informação

Atividade em grupo sobre legislação aplicada à tecnologia, incluindo análise de temas como:

* LGPD (Lei Geral de Proteção de Dados)
* Marco Civil da Internet
* Lei Brasileira de Inclusão
* Responsabilidade civil e ética digital
