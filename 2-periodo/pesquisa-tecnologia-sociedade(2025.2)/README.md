# Pesquisa, Tecnologia e Sociedade (2025.2)

## Descrição da Atividade
Esta atividade teve como objetivo realizar uma **pesquisa acadêmica utilizando a ferramenta Litmaps**, com o propósito de encontrar e analisar artigos científicos relacionados ao tema do projeto.

A pesquisa foi direcionada para o desenvolvimento de uma **solução digital voltada à gestão do Programa de Aquisição e Distribuição de Sementes**, analisando referências acadêmicas que pudessem apoiar o projeto.

## Ferramenta Utilizada
- **Litmaps** – plataforma utilizada para identificar e conectar artigos científicos relevantes, permitindo visualizar relações entre pesquisas e construir uma base teórica para o projeto.

## Objetivo da Pesquisa
- Encontrar artigos relacionados à **tecnologia aplicada à gestão pública e agricultura**.
- Identificar soluções digitais semelhantes ou conceitos que apoiem o desenvolvimento do sistema.
- Construir uma **base de referências científicas** para fundamentar o Projeto Integrador.

## Arquivos Disponíveis

📄 `apresentacao_pesquisa.pdf`  
Slides utilizados para apresentar os resultados da pesquisa.

📄 `artigos_relacionados.pdf`  
Documento contendo os artigos encontrados e suas conexões com o tema do projeto.

## Observação
Este trabalho foi desenvolvido **em grupo** e serviu como base teórica para apoiar o desenvolvimento do Projeto Integrador.
