# BDII – 3º Exercício: DQL com JOIN (Mini-mundo Petshop)

## Descrição da Atividade
Esta atividade foi proposta na disciplina **Banco de Dados II (BDII)**, ministrada pelo professor **Danilo Farias**.

O exercício teve como objetivo praticar consultas **SQL utilizando DQL (Data Query Language)** com **operações JOIN**, aplicadas ao mini-mundo do **Petshop**.

Foram utilizados os modelos **MER (Modelo Entidade-Relacionamento)** e **MR (Modelo Relacional)** fornecidos pelo professor, além dos scripts SQL contendo os comandos **DDL** (criação das tabelas) e **DML** (inserção de dados).

## Objetivo
Desenvolver consultas SQL capazes de gerar relatórios a partir do banco de dados do Petshop, utilizando **JOIN entre tabelas** para combinar informações relacionadas.

## Relatórios Desenvolvidos
Foram implementados **12 relatórios em SQL**, incluindo:

1. Empregados admitidos em determinado período.
2. Empregados com salário abaixo da média da empresa.
3. Departamentos com quantidade de empregados e médias salariais.
4. Quantidade total de vendas realizadas por empregado.
5. Vendas que envolveram prestação de serviços.
6. Histórico de serviços realizados para cada pet.
7. Vendas realizadas para clientes.
8. Serviços mais vendidos.
9. Formas de pagamento mais utilizadas.
10. Balanço diário de vendas.
11. Produtos e seus respectivos fornecedores.
12. Produtos mais vendidos.

## Arquivo da Atividade

📄 `relatorios_petshop_join.sql`  
Script SQL contendo as consultas **DQL com JOIN** utilizadas para gerar os 12 relatórios solicitados.

## Observação
Cada relatório foi desenvolvido utilizando **consultas SQL com JOIN**, permitindo relacionar dados entre diferentes tabelas do banco de dados do Petshop.
