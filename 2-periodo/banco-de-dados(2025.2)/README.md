# Banco de Dados (2025.2)

Esta pasta reúne as **atividades e materiais relacionados à disciplina de Banco de Dados**, desenvolvidos durante o segundo semestre do curso.

Aqui estão organizados os arquivos produzidos ao longo das atividades, incluindo modelos de dados, scripts SQL, diagramas e imagens utilizadas para representar a estrutura do banco de dados.

## Conteúdo da Pasta

Nesta pasta podem ser encontrados:

- **Modelos Conceituais**  
  Diagramas que representam as entidades, atributos e relacionamentos do sistema de forma abstrata.

- **Modelos Relacionais**  
  Representações das tabelas do banco de dados, incluindo chaves primárias e estrangeiras.

- **Scripts SQL**  
  Arquivos contendo comandos para criação e estruturação do banco de dados.

- **Diagramas e Imagens**  
  Representações visuais utilizadas para facilitar o entendimento da estrutura do banco.

## Objetivo

As atividades desta pasta têm como objetivo aplicar conceitos fundamentais de **modelagem e desenvolvimento de bancos de dados**, incluindo:

- Estruturação de dados
- Definição de relacionamentos
- Modelagem conceitual e relacional
- Implementação utilizando SQL

Esses materiais também contribuem para o desenvolvimento do **Projeto Integrador**, no qual o banco de dados é utilizado para estruturar e organizar as informações do sistema proposto.
