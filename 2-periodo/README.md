# 2º Período

Esta pasta reúne as atividades e projetos desenvolvidos durante o **2º período do curso de Tecnologia da Informação no SENAC**.

## 📚 Conteúdos estudados

Neste período foram aprofundados conceitos relacionados ao desenvolvimento de sistemas e organização de dados.

## 📂 Estrutura

As atividades estão organizadas por **disciplina**, contendo:

* exercícios práticos
* projetos acadêmicos
* códigos e scripts desenvolvidos nas aulas
* documentações de atividades

Cada disciplina possui sua própria pasta para melhor organização dos conteúdos.

## 🎓 Contexto

Os materiais presentes nesta pasta representam a continuidade da formação acadêmica, com foco no desenvolvimento de habilidades técnicas na área de tecnologia.
