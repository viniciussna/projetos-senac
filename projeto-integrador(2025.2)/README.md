# Projeto Integrador – 2º Semestre (2025.2)

## Atividade em grupo
Esta atividade foi realizada em grupo e consiste no desenvolvimento do **Projeto Integrador** do 2º semestre, contendo:

- Protótipo das páginas da aplicação (Figma / FigJam)
- Estrutura do banco de dados utilizado no projeto

## Conteúdo da pasta

### prototipo/
Contém todas as telas do protótipo do PI, incluindo fluxo de navegação, wireframes e design responsivo.

### banco-dados/
Contém diagramas, scripts SQL e exportações do banco de dados utilizadas no projeto.

## Observações
O projeto aborda a integração entre frontend e backend, buscando representar a aplicação de forma funcional e estruturada para testes e apresentação final.
