# 1º Período

Esta pasta contém atividades e projetos desenvolvidos durante o **1º período do curso de Tecnologia da Informação no SENAC**.

## 📚 Conteúdos estudados

Durante este período foram abordados conceitos introdutórios da área de tecnologia, incluindo fundamentos importantes para as disciplinas seguintes.

## 📂 Estrutura

Dentro desta pasta estão organizadas as atividades por **disciplina**, podendo incluir:

* exercícios práticos
* projetos acadêmicos
* códigos desenvolvidos em aula
* materiais complementares

Cada disciplina possui sua própria pasta contendo os arquivos e documentações necessárias.

## 🎓 Contexto

Todos os materiais presentes nesta pasta fazem parte das atividades acadêmicas realizadas durante o primeiro período do curso.
