# Currículo em HTML e CSS

Este projeto foi desenvolvido como atividade da disciplina **Coding (2026.1)** no SENAC.

A proposta da atividade foi criar um **currículo simples utilizando HTML e CSS**, com o objetivo de praticar a estrutura básica de uma página web e aplicar estilos utilizando CSS.

## 📄 Conteúdo do Projeto

O projeto contém:

* Estrutura da página utilizando **HTML**
* Estilização utilizando **CSS**
* Informações organizadas no formato de **currículo**

## 📦 Arquivo do Projeto

O projeto está disponível em formato **.zip**, contendo todos os arquivos necessários para visualizar o currículo no navegador.

Para acessar:

1. Baixe o arquivo `.zip`
2. Extraia os arquivos
3. Abra o arquivo `index.html` no navegador

## 🛠 Tecnologias utilizadas

* HTML
* CSS

## 🎓 Contexto acadêmico

Atividade realizada durante a primeira aula da disciplina **Coding – 2026.1**, com foco em introdução ao desenvolvimento web.
